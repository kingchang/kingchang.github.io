Gitlab CD/CD

若使用 gitlab-cicd 時，需要下列3個步驟

  1. 建立 .gitlab-ci.yml 檔案
stages:
  - package

package-job:
  - stage: package
  - tags:
      - aaa
      - bbb
  - script: ./gradlew build
  1. Click Setting -> [CI/CD Pinelines]
  2. 在 執行 gitlab-ruuner register 時將 url 跟 token 註冊進去，執行完成後，修改 config 檔案，加入 clone_url = “http://gitlab 內網ip”

若有 .gitlab-ci.yml 有使用 tag 時，需將所有 tag 加入才能正常運作
